📋 Setup Instructions

  1. Open Google Sheets → Create a new sheet named "Thryve Spark Database"
  2. Go to Extensions → Apps Script
  3. Delete any existing code and paste the entire contents of ThryveSparkBackend.gs
  4. Run the _setup() function once — grant all permissions when prompted
  5. Click Deploy → New deployment → Web app
  6. Set: Execute as Me · Who has access: Anyone
  7. Copy the Web App URL and paste it above → click Save
💡 The Apps Script backend handles: link creation, one-time validation, Sheet logging (Links + Responses tabs), PDF generation, student confirmation email, and counselor PDF report email — all automatically on submission. The counselor always shares the link manually.
